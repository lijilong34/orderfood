# 御膳房点餐系统
#####
## 项目介绍
###
御膳房点餐系统是基于java+vue3开发的智能点餐系统
## 功能介绍
1. 支持查看排名前三十的商品和店家
2. 支持分类查询
3. 支持分类查询
4. 支持多种支付方式(余额、支付宝)
5. 支持短信验证码服务
6. 加入滑块验证
7. 支持营养大师(1.语音输入 2.上传图片分析 3.使用ai大模型，图片识别的有调用限制,支持操作浏览器点餐)
